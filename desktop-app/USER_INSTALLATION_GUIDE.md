# Shinde Tractors Attendance System - Installation Guide

## For End Users

### System Requirements
- Windows 7 or later (Windows 10/11 recommended)
- 200 MB free disk space
- Internet connection (for database access)

### Installation Steps

1. **Download the Installer**
   - Get the file: `Shinde Tractors Attendance Setup 1.0.0.exe`

2. **Run the Installer**
   - Double-click the downloaded file
   - If Windows shows a security warning, click "More info" then "Run anyway"

3. **Choose Installation Location**
   - Default: `C:\Program Files\Shinde Tractors Attendance`
   - Or click "Browse" to choose another location

4. **Review License Agreement**
   - Read and accept the license terms

5. **Complete Installation**
   - Click "Install"
   - Wait for installation to complete
   - Check "Run Shinde Tractors Attendance" to launch immediately

### First Launch

1. **Find the App**
   - Desktop: Look for "Shinde Tractors Attendance" icon
   - Start Menu: Search for "Shinde Tractors Attendance"

2. **Login**
   - Enter your credentials provided by your administrator
   - Default login page will appear

### Features Available

- ✅ Mark Daily Attendance
- ✅ Manage Employees
- ✅ Set Working Days
- ✅ Calculate Salaries (with Sunday compensation)
- ✅ Generate Reports
- ✅ Export to Excel

### Troubleshooting

#### App Won't Start
1. Ensure you have internet connection
2. Check with administrator that database is accessible
3. Try restarting your computer

#### Can't Login
1. Verify your credentials with administrator
2. Check internet connection
3. Ensure database URL is correct

#### Need to Move the App
1. Uninstall from current location
2. Reinstall in desired location

### Uninstallation

To remove the application:

1. **Method 1: Settings**
   - Open Windows Settings
   - Go to Apps > Installed Apps
   - Find "Shinde Tractors Attendance"
   - Click "Uninstall"

2. **Method 2: Control Panel**
   - Open Control Panel
   - Programs > Uninstall a program
   - Select "Shinde Tractors Attendance"
   - Click "Uninstall"
